

# Dev-tionary: 개발 영어 사전

개발자를 위한 영어 용어 위키 — 발음, 뜻 풀이, 예문을 한국어로 제공하는 웹앱

## 페이지 구성

### 1. 홈페이지
- 다크모드 기본, 코드 에디터 느낌의 모던 UI
- 큰 검색바 (자동완성 포함)로 용어 빠르게 탐색
- 카테고리별 브라우징: JavaScript/TypeScript, Python, Java, Git, CLI, SQL, 공식문서 표현, 협업 표현
- "오늘의 단어" 또는 랜덤 용어 하이라이트 섹션

### 2. 용어 상세 페이지
- **영어 단어** + IPA 발음기호 + 한글 발음 표기
- **🔊 발음 듣기 버튼** (브라우저 Web Speech API TTS)
- **뜻 풀이**: 원래 영어 뜻 → 개발에서의 의미 연결 설명
- **약어 풀이**: var → variable, const → constant 등
- **예문**: 공식 문서에서 추출한 실제 사용 예시 + 한국어 번역
- **카테고리 태그** 및 관련 용어 링크

### 3. 카테고리 목록 페이지
- 카테고리별 용어 그리드/리스트 뷰
- 알파벳순, 최신순 정렬

### 4. 검색 결과 페이지
- 실시간 검색 (영어/한글 모두 지원)
- 카테고리 필터

## 데이터베이스 (Supabase)

### terms 테이블
- 영어 단어, IPA 발음기호, 한글 발음, 약어 원형(있는 경우)
- 영어 원래 뜻, 개발 맥락 뜻 풀이 (한국어)
- 카테고리, 예문(들), 예문 번역
- 관련 용어 링크

### categories 테이블
- 카테고리명, 아이콘, 설명

## 디자인
- 다크모드 기본 (라이트모드 토글 가능)
- 모노스페이스 폰트 활용, 코드 블록 스타일 카드
- 깔끔한 타이포그래피 중심 레이아웃
- 반응형 모바일 지원

## 다국어 확장 구조
- 뜻 풀이/번역을 locale 기반으로 분리 가능한 구조로 설계
- 우선 한국어로 시작, 추후 일본어/중국어 등 확장 가능

## 초기 샘플 데이터
- 약 20~30개 핵심 용어 시드 데이터 포함 (branch, coalesce, var, const, func, concat, merge, pull, push, deploy, commit 등)
